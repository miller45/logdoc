Benutzung von less Variablen für Farben und Schriftgrößen

Wir "compilieren" unsere css Dateien aus so genannten less-dateien (siehe http://lesscss.org ). Das ermöglicht es uns zentrale Angaben wie Schriftfarben und -Größen leichter wiederzuverwenden. Ausserdem lassen sich mit less Berechnungen durchführen die von reinen css dateien nicht( oder noch nicht in allen Browsern) unterstützt werden. Dabei sind die Angaben in form von zentralen Variablen in der Datei @oi-variables.less hinterlegt. Dadurch kann diesen Schriftdefinition an verschiedenen Stellen wiedervwendet werden,

 

Beispiele für less Variablen

Hinweis: Vor jeder variable steht @oi- Das @ "bezeichnet" eine less Variable. Das oi- kommt von objectinformation. Dieser @oi- Prefix ist nötig um Konflikte mit Bootstrap less Variablen zu vermeiden (dort gibt es z.B. schon @font-size-h1)

Auszug aus @oi-variables.less
@oi-regular-font: "Segoe UI","SegoeUI-Regular",Arial,sans-serif;
Auszug aus @main.less
.objectinformation p {
    font-family: @oi-regular-font;
    font-size: @oi-font-size-base;
    color: @oi-regular-color;
}

.objectinformation h6, .objectinformation .h6 {
    font-family: @oi-regular-font;
}

Technische Details

Technische Hintergründe für die Einbindung/Benutzung der css/less Dateien

Webfonts

An dem oben gezeigten Beispiel von @oi-regular-font sieht man die Angabe Segoe UI und Segoe-UI-Regular

Die erste Angabe führt beim Internet Explorer dazu, dass die im System installierte Schrift von Segoe UI benutzt wird. Bei andere Browsern z.B. Chrome(=ONE Embedded) oder Systemem z.B. Smartphones/iPad kann diesen Schrift z.T. nur über sogenannte webfonts eingebunden werden. Dafür haben wir im bsweb Projket eine webfonts.less Datei , die über @font-face die Schrift Segoe UI über den Name Segoe-UI-Regular einbindet.
